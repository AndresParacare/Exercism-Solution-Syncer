# 🦀 Rust Exercises & Practice

¡Bienvenido a mi repositorio de ejercicios de Rust! Este espacio está dedicado a documentar mi proceso de aprendizaje, resolver retos de código y dominar los conceptos clave de este lenguaje de programación.

---

## 🚀 Contenido del Repositorio

Los ejercicios están organizados por niveles de complejidad o por conceptos clave del lenguaje. Aquí tienes un mapa de lo que puedes encontrar:

### 🔰 Conceptos Básicos
* **Hola Mundo:** Familiarización con la sintaxis básica y `macro_rules!`.
* **Variables y Tipos:** Mutabilidad, tipos de datos primitivos y *shadowing*.
* **Estructuras de Control:** Uso de `if`, `else`, `loop`, `while` y el poderoso `match`.

### 🧠 Conceptos Intermedios
* **Ownership y Borrowing:** Ejercicios prácticos para entender las reglas de pertenencia, referencias y tiempos de vida (*lifetimes*).
* **Structs y Enums:** Modelado de datos y uso de patrones avanzados.
* **Manejo de Errores:** Prácticas con `Result` y `Option` sin abusar de `unwrap`.

### ⚙️ Avanzado y Proyectos
* **Concurrencia:** Hilos (*threads*), paso de mensajes y estado compartido.
* **Smart Pointers:** Uso de `Box`, `Rc`, `Arc` y `RefCell`.
* **Proyectos pequeños:** Implementaciones de CLI, algoritmos o emuladores sencillos.
